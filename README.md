# 📝 Todo Redux App

A simple and efficient Todo application built using **React** and **Redux Toolkit**. This project demonstrates modern state management using slices, actions, and reducers while maintaining a clean and responsive UI.

---

## 🚀 Features

* ➕ Add new todos
*   Remove existing todos
* ⚡ Fast and optimized state management with Redux Toolkit
* 🎯 Clean and minimal UI
* 🔄 Real-time updates using React hooks

---

## 🛠️ Tech Stack

* **React.js**
* **Redux Toolkit**
* **React-Redux**
* **JavaScript (ES6+)**
* **Tailwind CSS** (for styling)

## ⚙️ Installation & Setup

1. Clone the repository:

```
git clone https://github.com/guri5405/todoredux.git
```

2. Navigate to project folder:

```
cd todoredux
```

3. Install dependencies:

```
npm install
```

4. Run the app:

```
npm run dev
```

---

## 🧠 Concepts Covered

* Redux Store configuration
* Creating slices using `createSlice()`
* Dispatching actions using `useDispatch()`
* Accessing state using `useSelector()`
* Immutable state updates with Redux Toolkit
